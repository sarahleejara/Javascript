Exercices: Variables

Exercice: Diseur de bonne aventure

Déclarez les variables suivantes:

  • nombre d'enfants
  • nom d'une femme
  • un pays
  • métier

Solution

var enfants = Number(2);
var femme = "Anna";
var pays = "Bahamas";
var metier = "blogger";
console.log("Vous serez " + metier + " habiterez à " + pays + ", et marié a " + femme + " avec " + enfants + " enfants.")
